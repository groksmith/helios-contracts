# Helios Contracts Forge Project

## Prerequisites 

Install Just
https://github.com/casey/just

Install Foundry
https://book.getfoundry.sh/getting-started/installation

Install LCOV
https://formulae.brew.sh/formula/lcov

Install Slither
```pip3 install slither-analyzer --user```

Install Echidna
```brew install echidna```
